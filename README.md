# Food Deli
website การขายอาหาร และมีระบบจัดการสินค้า และระบบติดตามสถานะการจัดส่ง
# Available Scripts

## Frontend
### `cd frontend`
### `npm install`
### `npm run dev`

## Backend
### `cd backend`
### `npm install`
### `npm run server`

## Admin Side
### `cd admin`
### `npm install`
### `npm run dev`
